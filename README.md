## 🚀 Getting Started

### Prerequisites

You’ll need:

* A **C Compiler** like `gcc` or `clang`
* A **terminal or code editor** (VS Code, Code::Blocks, or Dev-C++)
* Basic understanding of how to compile and run C programs

### Run Any Program

```bash
gcc filename.c -o output
./output
```

Example:

```bash
gcc hello_world.c -o hello
./hello
```
